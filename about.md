# About Reter Code

AI-powered code reasoning MCP server.

## What It Does

Reter Code gives Claude Code access to:

- **Logical Reasoning** - RETER engine for code understanding
- **Multi-Language Analysis** - Python, C#, C++, JavaScript
- **Session Persistence** - Track thoughts, requirements, tasks
- **UML Diagrams** - Class hierarchy, sequence, dependencies
- **Refactoring** - Recommendations and test coverage analysis

## Tools

| Category | Tools |
|----------|-------|
| Thinking & Reasoning | `thinking`, `session`, `items`, `project` |
| Code Analysis | `code_inspection`, `recommender`, `diagram` |
| Instance Management | `instance_manager` |

## Supported Languages

| Language | Extensions |
|----------|------------|
| Python | `.py` |
| C# | `.cs` |
| C++ | `.cpp`, `.hpp`, `.h`, `.cc` |
| JavaScript | `.js` |

## Features

- Logical reasoning with RETER engine
- Multi-language code analysis
- Session-based thinking with persistence
- Requirements and task tracking
- UML diagram generation
- Refactoring recommendations
- Test coverage analysis
